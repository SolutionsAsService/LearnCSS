What is the CSS Box Model?

The CSS box model is a fundamental concept that describes how elements are structured and how they interact with each other on a web page. It consists of four components: content, padding, border, and margin.

Components of the Box Model

  • Content: The innermost part where text and images appear.
  • Padding: The space between the content and the border. It clears an area around the content.
  • Border: The area surrounding the padding. It can be styled with different colors, widths, and patterns.
  • Margin: The outermost space that separates the element from other elements. It clears an area outside the border.

Visualizing the Box Model

Example of Box Model Properties


div {
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 10px solid black;
    margin: 30px;
}

In this example, the element will have a total width of 360px (300px + 2 * 20px padding + 2 * 10px border) and a total height of 280px (200px + 2 * 20px padding + 2 * 10px border).
