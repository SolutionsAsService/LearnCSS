What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It allows you to create complex and responsive layouts with ease, using rows and columns to define the structure of your content.

Key Concepts of CSS Grid

  • Grid Container: The parent element that contains grid items. It is defined by setting the display property to grid or inline-grid.
  • Grid Items: The child elements of a grid container. They are placed into the grid using various grid properties.

Common CSS Grid Properties

Grid Container Properties

  • display: grid;
  • grid-template-columns: Defines the columns of the grid.
  • grid-template-rows: Defines the rows of the grid.
  • grid-gap: Defines the gap between grid items.
  • justify-items: Aligns grid items along the row axis.
  • align-items: Aligns grid items along the column axis.

Grid Item Properties

  • grid-column: Specifies the column start and end positions for the grid item.
  • grid-row: Specifies the row start and end positions for the grid item.
  • justify-self: Aligns the grid item within its column.
  • align-self: Aligns the grid item within its row.

Example of CSS Grid


.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

.item {
    background-color: lightgray;
    padding: 20px;
    text-align: center;
}
